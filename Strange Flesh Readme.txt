Thanks for opening this file and reading it! You're awesome!

Strange Flesh is a sexy, gay, adult beat 'em up where you play 
the role of The Bartender, a master of the hypnotic arts and a 
servant of the bar, Strange Flesh.

In the game, you'll infiltrate the mind of one of the bar's many 
troubled patrons, and help him out however you see fit.

			      --- CONTROLS ----

The game has no mouse controls. It is controlled entirely with the keyboard.

          Arrows/WASD          Walk, Run, Select menu options
                Enter          Pause, Activate menu options
                    J          Punch, Activate menu options
                    K          Smoke, back on menus
                    L          Grab
                Space          Jump

Some basic controller support is available in the settings menu in the game.
You can play with that, but we were unable to make it work reliably in testing.
If it works for you and you like it, congrats! If not, then that's ok. The 
game was designed and tested using the keyboard. It's plenty fun that way.

For more information, check out the PDF manual or, heck, 
just play the game and see what happens. Enjoy!
